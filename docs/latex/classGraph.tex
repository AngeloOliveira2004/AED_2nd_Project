The diameter of the graph. 
\end{DoxyReturn}

Time Complexity\+: O(V$^\wedge$2 + V$\ast$E)
\begin{DoxyItemize}
\item V is the number of vertices in the graph.
\item E is the number of edges in the graph.
\item The function consists of two nested loops, where the outer loop runs V times (once for each vertex) and the inner loop runs BFS, which takes O(V + E) time.
\item The overall complexity is O(V$^\wedge$2 + V$\ast$E).
